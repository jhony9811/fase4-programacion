# fase4-programacion
Sistema de clientes con manejo de excepciones
## Descripción
Proyecto desarrollado en Python aplicando programación orientada a objetos, validación de datos y manejo de excepciones.

El sistema permite registrar clientes controlando errores mediante estructuras try/except y excepciones personalizadas.

Además, implementa un archivo de registro llamado errores.log donde se almacenan los errores generados durante la ejecución del programa.

## Características
- Registro de clientes
- Validación de nombre
- Validación de identificación
- Validación de correo electrónico
- Validación de teléfono
- Manejo de excepciones
- Registro de errores en archivo log
- Trabajo colaborativo con GitHub

## Tecnologías utilizadas
- Python
- Visual Studio Code
- GitHub
